Guide des paramètres

🎯 Objectif Prod : Définit le nombre total d'échelles à produire. Impacte les actions configurées en "/ Grille" ou "/ Échelle".

🧇 Gâteaux / Grille : Permet de déduire les actions. Ex: Si 1 grille = 30 gâteaux et 1 action = 10 gâteaux, l'action est répétée 3 fois/grille.

📦 Gâteaux / Palette : Permet de déduire les actions. Ex: Si 1 palette = 1000 gâteaux et 1 action = 10, l'action est répétée 100 fois/palette.

🔄 Répétition : Multiplie automatiquement vos données selon la logique :

  • / Jour : L'action est comptée exactement le nombre de fois saisi.
  • / Grille : Utilisera "Gâteaux/Grille", multiplié par 17 (grilles/échelle) et par l'Objectif d'Échelles.
  • / Échelle : La valeur saisie est multipliée par l'Objectif d'Échelles.
  • / Palette : Utilisera "Gâteaux/Palette" divisé par l'objectif par action.
